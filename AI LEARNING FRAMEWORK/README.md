# AI LEARNING FRAMEWORK

Framework untuk **belajar topik / skill baru** secara terstruktur dengan AI sebagai tutor + sparring partner. Cocok untuk: bahasa pemrograman baru, domain teknis (DSP, kriptografi, kompiler), domain non-teknis (mixing, hukum kontrak, statistik bayesian).

## Prinsip
1. **Outcome-first.** Belajar untuk *bisa apa*, bukan "tahu tentang".
2. **Active over passive.** Catat, latihan, recall — bukan baca / nonton terus.
3. **Project as proof.** Hasil akhir bukan sertifikat, tapi sebuah project yang jalan / karya nyata.
4. **Spaced repetition.** AI bantu generate flashcard / quiz dari catatan.
5. **Public learning > private learning** (kalau bisa). Tulis blog / share, dapat feedback.

## Stages

```
[01-Goal] ──► [02-Topic Map] ──► [03-Plan] ──► [04-Active] ──► [05-Project] ──► [06-Retro]
```
