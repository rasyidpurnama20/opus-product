# AI WRITING FRAMEWORK

Framework untuk **menulis konten** secara terstruktur — blog post teknis, dokumentasi internal, proposal, technical reports, thread media sosial, dan tulisan panjang lainnya. AI bantu di setiap stage tanpa mengambil alih voice Anda.

## Prinsip
1. **Audience first.** Tulis untuk satu pembaca spesifik, bukan "semua orang".
2. **Outline sebelum draft.** Tulisan tanpa outline biasanya berputar-putar.
3. **Edit ≠ proofread.** Edit untuk struktur & klarifikasi; proofread untuk typo.
4. **Voice manusia, bukan voice AI.** Pakai AI untuk drafting, tapi rewrite agar tetap suara Anda.
5. **Iterasi pendek > marathon.** Pomodoro 25 menit, lalu jeda.

## Stages
```
[01-Audience] ──► [02-Outline] ──► [03-Draft] ──► [04-Edit] ──► [05-Polish] ──► [06-Publish]
```

| # | Stage | Output | Code |
|---|---|---|---|
| 01 | Audience | Audience Brief | `AB` |
| 02 | Outline | Outline | `OL` |
| 03 | Draft | First Draft | `FD` |
| 04 | Edit | Edit Pass | `EP` |
| 05 | Polish | Polish Pass | `PP` |
| 06 | Publish | Publish Plan | `PB` |
