# AI DECISION FRAMEWORK

Framework untuk **mengambil keputusan penting** secara terstruktur — teknis (pilih database, framework), karir (terima offer atau tidak), produk (rilis sekarang atau tunda), pribadi (pindah kerja/kota).

AI bantu generate opsi & analisis trade-off, tapi **keputusan tetap di Anda**. Framework ini mencegah dua kegagalan klasik: (1) memutuskan terlalu cepat dengan opsi terbatas, (2) menunda terus dengan analysis paralysis.

## Prinsip
1. **Reversible vs irreversible.** Jangan over-engineer keputusan reversible; jangan rush keputusan irreversible.
2. **Time-box.** Setiap stage punya deadline. Analysis paralysis = no decision.
3. **3+ options minimum.** Jika hanya 2 (yes/no), generate ekstrem ke-3 untuk recalibrate.
4. **Premortem before commit.** "Bayangkan ini gagal — kenapa?"
5. **Decide, don't dilute.** Keputusan harus tegas, bukan "let's see how it goes".

## Stages
```
[01-Frame] ──► [02-Options] ──► [03-Criteria] ──► [04-Evaluate] ──► [05-Decide] ──► [06-Review]
```

| # | Stage | Output | Code |
|---|---|---|---|
| 01 | Frame | Decision Brief | `DB` |
| 02 | Options | Options Map | `OM` |
| 03 | Criteria | Decision Criteria | `DC` |
| 04 | Evaluate | Trade-off Matrix | `TM` |
| 05 | Decide | Decision Record | `DR` |
| 06 | Review | Decision Retrospective | `DRT` |
